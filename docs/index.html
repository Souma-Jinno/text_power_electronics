本書の理念（企画書より）

  1. 素子が「なぜそう動くか」から機能を導く（読める→作れるへ）
  2. 「エネルギーをどう蓄え、どう放出するか」の一貫視点で全変換回路を串刺し
  3. 仮定（理想スイッチ・定常状態・小リプル）を毎回明示
  4. 目標リプル→L・C値という設計手順まで下ろす
  5. LTspiceで全回路を再現可能に
