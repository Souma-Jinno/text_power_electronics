\chapter{電力変換で生じる電磁ノイズとEMC}
\label{chap11}

本書はここまで一貫して「スイッチングは速いほど良い」という方向を向いてきた。
しかし，速いスイッチングはただでは手に入らない。急峻に切り替わる電圧・電流は豊富な高周波成分を含み，
それが配線の寄生成分を通って回路の外へ漏れ出すと，周囲の機器の動作を乱す\textbf{電磁ノイズ}になる。
現代の電源設計では，効率や小型化と同じ重みで「ノイズを規格値以下に抑えること」が要求される。
本章はこの問題を，勘や経験則ではなく，本書で学んだ物理と回路の言葉で定量的に扱う。

本書で学んだ高速スイッチングの代償を，素子の物理（第I部）と変換回路（第II部）の言葉で理解し，対策する。

\section{高周波化による小型化とトレードオフ}
\label{sec11.1}

\subsection{5章の宿題 ― 部品は小さくなるがノイズが増える}

5章で導いた降圧チョッパの設計式を思い出そう。必要なインダクタンスとキャパシタンスは
\begin{equation}
L = \frac{(V_{\mathrm{in}} - V_{\mathrm{out}})\, D}{f\, \mathit{\Delta}I_L}，\qquad
C = \frac{\mathit{\Delta}I_L}{8 f\, \mathit{\Delta}V_{\mathrm{out}}}
\label{eq11.1}
\end{equation}
であり，どちらも分母にスイッチング周波数$f$をもつ。同じリプルを許すなら，$f$を10倍にすれば$L$も$C$も10分の1で済む。
これを可能にしたのが3章のSiC・GaNである。寄生容量が小さいこれらの素子は，
オン・オフの切替えに要する\index{たちあがりじかん@立上り時間}\textbf{立上り時間}$t_r$が短く
スイッチング損失が小さいため，高い周波数で運転できる。

ところが\FIGref{fig11.1}に示すとおり，話はそれで終わらない。
スイッチング波形は矩形波であり，基本波（周波数$f$）の整数倍の周波数の
\index{こうちょうは@高調波}\textbf{高調波}を豊富に含む。
$f$を10倍にするとスペクトル全体が高い側へ10倍移動し，
後で示すとおり同じ周波数で比べたノイズ成分は10倍（$+20$\,dB）大きくなる。
\textbf{高周波化は，部品を小さくするのと同じ倍率で高周波ノイズを増やす} ―
これが本章を貫くトレードオフである。

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.1.eps}
\end{center}
\caption{高周波化のトレードオフ。(a)必要な$L$・$C$は$1/f$で小さくなるが，
(b)スイッチング波形の高調波の包絡線は持ち上がり，同じ周波数で比べると$f$が10倍で$20$\,dB増える}
\label{fig11.1}
\end{figure}

\begin{定義}[電磁ノイズ]
\label{def11.1}
回路の動作にともなって発生し，設計者が意図しない経路で伝わって他の回路や機器の動作を乱す
不要な電圧・電流・電磁場の成分を\index{でんじのいず@電磁ノイズ}\textbf{電磁ノイズ}
（\index{いーえむあい@EMI}\textbf{EMI}：electromagnetic interference，電磁妨害）という。
電源線や信号線などの導体を伝わるものを\index{でんどうのいず@伝導ノイズ}\textbf{伝導ノイズ}，
電磁波として空間を伝わるものを\index{ほうしゃのいず@放射ノイズ}\textbf{放射ノイズ}という。
\end{定義}

\subsection{なぜ高い周波数が問題なのか ― 波長と回路の大きさ}

問題は周波数が上がって\index{はちょう@波長}\textbf{波長}$\lambda = c_0/f$
（$c_0 \approx 3\times10^8$\,m/sは光速）が配線の物理的な長さに近づいたときである。
導体の長さが波長の4分の1程度になると，その導体は効率のよいアンテナとして働き，
伝導ノイズが放射ノイズに化ける。商用周波数60\,Hzの波長は5000\,kmもあるが，
$100$\,MHzでは3\,m ― 1\,mの電源ケーブルは立派なアンテナである。
スイッチングの高調波がこの帯域まで届くかどうかを決めるのが，次節のスペクトルの話である。

\section{スイッチング波形のフーリエ級数}
\label{sec11.2}

\subsection{台形波のスペクトル}

ノイズ源の正体を定量化しよう。実際のスイッチング波形は，理想の矩形波ではなく，
有限の立上り時間$t_r$・立下り時間$t_f$をもつ\index{だいけいは@台形波}\textbf{台形波}である
（\FIGref{fig11.2}(a)。以下$t_r = t_f$とする）。
振幅$A$，周期$T$，パルス幅$\tau$（$= DT$）の台形波を
\index{ふーりえきゅうすう@フーリエ級数}\textbf{フーリエ級数}に展開すると，
$n$次高調波（周波数$f_n = n/T$）の振幅は
\begin{equation}
a_n = 2A\, \frac{\tau}{T}
\left| \frac{\sin (\pi f_n \tau)}{\pi f_n \tau} \right|
\left| \frac{\sin (\pi f_n t_r)}{\pi f_n t_r} \right|
\label{eq11.2}
\end{equation}
となる（導出はフーリエ級数の教科書に譲る）。
形は複雑に見えるが，$\sin x / x$という同じ形の因子が2つ並んでいるだけであり，
1つめはパルス幅$\tau$が，2つめは立上り時間$t_r$が決める。
この関数の性質は2つだけ覚えればよい。$x$が小さいときは$\sin x \approx x$だから
$|\sin x / x| \approx 1$，$x$が大きいときは$|\sin x| \le 1$だから
\begin{equation}
\left| \frac{\sin x}{x} \right| \le \frac{1}{x}
\label{eq11.3}
\end{equation}
と，上限が$1/x$で減っていく。2つの振舞いの境目は，ちょうど$1/x = 1$となる$x = 1$である。
式\ref{eq11.2}の1つめの因子では$x = \pi f \tau$，2つめの因子では$x = \pi f t_r$だから，境目の周波数はそれぞれ
\begin{equation}
f_1 = \frac{1}{\pi \tau}，\qquad f_2 = \frac{1}{\pi t_r}
\label{eq11.4}
\end{equation}
である。$t_r \ll \tau$だから必ず$f_1 < f_2$になる。
すなわち高調波の振幅の上限（\index{ほうらくせん@包絡線}\textbf{包絡線}）は，
2つの\index{おれてんしゅうはすう@折れ点周波数}\textbf{折れ点周波数}$f_1$，$f_2$をもつ折れ線になる。

\begin{screen}
\noindent\textbf{台形波スペクトルの包絡線（\FIGref{fig11.2}(b)）}\\[3pt]
\begin{enumerate}
\item $f < f_1 = 1/\pi\tau$：ほぼ一定値$2AD$
\item $f_1 < f < f_2 = 1/\pi t_r$：周波数10倍ごとに振幅$1/10$（$-20$\,dB/dec）で減少
\item $f > f_2$：周波数10倍ごとに振幅$1/100$（$-40$\,dB/dec）で急減
\end{enumerate}
第1の折れ点はパルス幅（周期）が，第2の折れ点は立上り時間が決める。
$f_2$より上では高調波は急速に消えるから，
\textbf{ノイズとして効くのはおよそ$f_2 = 1/\pi t_r$までの帯域}である。
\end{screen}

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.2.eps}
\end{center}
\caption{台形波とそのスペクトル。包絡線は$f_1 = 1/\pi\tau$まで平坦，$f_2 = 1/\pi t_r$まで
$-20$\,dB/dec，それ以上は$-40$\,dB/dec（$f = 100$\,kHz，$D = 0.5$，$t_r = 10$\,nsの場合）}
\label{fig11.2}
\end{figure}

\subsection{周期と立上り時間がスペクトルを決める}

式\ref{eq11.4}から，本章の冒頭で述べたトレードオフが数式で読める。
\textbf{スイッチング周波数$f$を10倍にする}と$\tau = DT$が$1/10$になり，$f_1$が10倍の周波数へ移る。
包絡線の平坦部の高さ$2AD$は変わらないから，$-20$\,dB/dec領域の包絡線は
全体として10倍（$+20$\,dB）持ち上がる（\ref{fig11.1}(b)）。
また\textbf{立上り時間$t_r$を$1/10$にする}と$f_2$が10倍の周波数へ移り，
それまで$-40$\,dB/decで急減していた帯域が$-20$\,dB/decの緩い減衰に置き換わる。
SiC・GaNの高速スイッチングはまさにこの$t_r$を短くする技術であり，
損失の面では望ましい高速化が，スペクトルの面では
ラジオや無線通信が使う数十MHz以上の帯域へノイズを押し広げる。

\begin{例題}[折れ点周波数の計算]
\label{rei11.1}
スイッチング周波数$f = 100$\,kHz，デューティ比$D = 0.5$，立上り時間$t_r = 10$\,nsの台形波について，
2つの折れ点周波数$f_1$，$f_2$を求めよ。
\begin{解答}
パルス幅は$\tau = DT = 0.5 \times 10\,\mathrm{\mu s} = 5\,\mathrm{\mu s}$。式\ref{eq11.4}より
\begin{equation*}
f_1 = \frac{1}{\pi \times 5\times10^{-6}} \approx 64\,\mathrm{kHz}，\qquad
f_2 = \frac{1}{\pi \times 10\times10^{-9}} \approx 32\,\mathrm{MHz}
\end{equation*}
となる。高調波は緩い$-20$\,dB/decの減衰のまま，スイッチング周波数の数百倍の帯域まで居座るのである。
\end{解答}
\end{例題}

\begin{注意}
包絡線はあくまで振幅の\textbf{上限}である。個々の高調波は$\sin$のゼロ点で細かく落ち込むし，
$D = 0.5$のときは偶数次の高調波が消える。しかしノイズの見積りでは
「最悪どこまで大きくなり得るか」が知りたいのだから，上限を与える包絡線で考えるのが実用的である。
\end{注意}

\begin{問}
\item ある電源のスイッチ素子をSi（$t_r = 40$\,ns）からGaN（$t_r = 4$\,ns）に置き換えた。
第2の折れ点周波数$f_2$はいくらからいくらへ移るか。
また周波数$100$\,kHz，$30$\,MHz，$300$\,MHzの電磁波の波長をそれぞれ求め，
$f_2$付近の高調波にとって$1$\,mの電源ケーブルがアンテナになり得ることを確かめよ。
\end{問}

\section{デシベルと電磁ノイズの評価}
\label{sec11.3}

\subsection{桁で暴れる量は対数で扱う}

ノイズのスペクトルは数kHzから数百MHzまで，振幅にして数万倍の範囲で変化する。
そこで電磁ノイズの世界では，振幅を対数の単位\index{でしべる@デシベル}\textbf{デシベル}（dB）で表す。

\begin{定義}[デシベル]
\label{def11.2}
基準の電力$P_{\mathrm{ref}}$に対する電力$P$の比をデシベルで表した値は
\begin{equation}
G_P = 10 \log_{10} \frac{P}{P_{\mathrm{ref}}}\quad \mathrm{[dB]}
\label{eq11.5}
\end{equation}
である。同じ負荷$Z$にかかる電圧で表すと，$P = V^2/Z$だから
\begin{equation}
G_V = 10 \log_{10} \left( \frac{V}{V_{\mathrm{ref}}} \right)^{\!2}
= 20 \log_{10} \frac{V}{V_{\mathrm{ref}}}\quad \mathrm{[dB]}
\label{eq11.6}
\end{equation}
となる。\textbf{電力比は係数10，電圧比（電流比も）は係数20}である。
\end{定義}

掛け算がdBでは足し算になるので，「電圧2倍で$+6$\,dB，10倍で$+20$\,dB
（電力なら$+3$\,dBと$+10$\,dB）」さえ覚えれば組み合わせて暗算できる
（電圧40倍は$2\times2\times10$倍だから$6+6+20 = 32$\,dB）。

\subsection{ノイズの絶対値を表すdB$\mathrm{\mu}$V}

デシベルは「比」であるが，基準値を固定すれば絶対値の表現にもなる。
電磁ノイズの測定では，基準電圧を$1\,\mathrm{\mu V}$に固定した
\index{でぃーびーまいくろぼると@dB$\mathrm{\mu}$V}\textbf{dB$\mathrm{\mu}$V}が使われる。

\begin{定義}[dB$\mathrm{\mu}$V]
\label{def11.3}
電圧$V$をdB$\mathrm{\mu}$Vで表した値は
\begin{equation}
V\,[\mathrm{dB\mu V}] = 20 \log_{10} \frac{V}{1\,\mathrm{\mu V}}
\label{eq11.7}
\end{equation}
である。$1\,\mathrm{\mu V}$が$0$\,dB$\mathrm{\mu}$V，$1$\,mVが$60$\,dB$\mathrm{\mu}$V，
$1$\,Vが$120$\,dB$\mathrm{\mu}$Vに対応する。
\end{定義}

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.3.eps}
\end{center}
\caption{dBスケールの読み方。電圧の対数目盛とdB$\mathrm{\mu}$V表示の対応。
10倍ごとに$+20$\,dB，2倍ごとに$+6$\,dB進む}
\label{fig11.3}
\end{figure}

後述するEMC試験の規格値はdB$\mathrm{\mu}$Vで与えられる。測定値も対策部品の減衰量もdBで表しておけば
（\FIGref{fig11.3}），「ノイズ$66$\,dB$\mathrm{\mu}$V，規格$40$\,dB$\mathrm{\mu}$V，
よってフィルタで$26$\,dB落とす」と，掛け算・割り算がすべて足し算・引き算になる。

\begin{例題}[dB$\mathrm{\mu}$Vの計算]
\label{rei11.2}
電源線上のノイズ電圧を測定したところ，ある周波数で$2$\,mVであった。
(a)~この値をdB$\mathrm{\mu}$Vで表せ。
(b)~フィルタを挿入してノイズを$40$\,dB減衰させた。フィルタ通過後の値をdB$\mathrm{\mu}$Vと$\mathrm{\mu V}$で表せ。
\begin{解答}
(a) $2\,\mathrm{mV} = 2000\,\mathrm{\mu V}$だから，式\ref{eq11.7}より
\begin{equation*}
20 \log_{10} 2000 = 20 \log_{10} (2 \times 10^3) = 6 + 60 = 66\,\mathrm{dB\mu V}
\end{equation*}
（$\log_{10} 2 \approx 0.3$とした）。
(b) dBでは減衰は引き算だから$66 - 40 = 26\,\mathrm{dB\mu V}$。電圧に戻すと
$10^{26/20} = 10^{1.3} \approx 20\,\mathrm{\mu V}$である。
$40$\,dBの減衰が「$1/100$」を意味することを確かめてほしい。
\end{解答}
\end{例題}

\begin{注意}
係数10（電力）と20（電圧）の取り違えは，dB計算の間違いの大半を占める。
本章で扱うノイズのスペクトルや規格値は電圧（dB$\mathrm{\mu}$V）ベースだから，係数は常に20である。
\end{注意}

\begin{問}
\item 式\ref{eq11.7}について，対数の中身が無次元になっていること（単位の検算）を確かめよ。
また$V = 1$\,Vのとき$120$\,dB$\mathrm{\mu}$Vになることを検算せよ。
\end{問}

\section{配線が持つ寄生成分}
\label{sec11.4}

\subsection{回路図の「ただの線」は理想化である}

ノイズ源のスペクトルがわかった。次はそれが伝わる経路である。
回路図で素子と素子を結ぶ線は，インピーダンス$Z = 0$の理想の接続を表す記号にすぎない。
しかし実際の配線では，往復の電流がまわりに磁場をつくる（アンペールの法則）から，
配線は直列のインダクタンスをもつ。往復の導体間に電位差があれば電荷が現れ，
電気力線が張られる（ガウスの法則）から，並列のキャパシタンスももつ（\FIGref{fig11.4}）。
回路図に描いていないのに実物には必ず付いてくるこれらの成分を
\index{きせいせいぶん@寄生成分}\textbf{寄生成分}
（\index{きせいいんだくたんす@寄生インダクタンス}\textbf{寄生インダクタンス}・
\index{きせいきゃぱしたんす@寄生キャパシタンス}\textbf{寄生キャパシタンス}）という。

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.4.eps}
\end{center}
\caption{平行配線の寄生成分。(a)往復電流のつくる磁場が寄生インダクタンス$L'$として，
(b)導体上の電荷がつくる電場が寄生キャパシタンス$C'$として現れる}
\label{fig11.4}
\end{figure}

半径$a$の2本の導線を，表面間の距離$d$で平行に張った線路では，単位長さあたり
\begin{equation}
L' = \frac{\mu_0}{\pi} \ln \left( 1 + \frac{d}{a} \right)\quad \mathrm{[H/m]}
\label{eq11.8}
\end{equation}
\begin{equation}
C' = \frac{\pi \varepsilon_0}{\ln \left( 1 + \dfrac{d}{a} \right)}\quad \mathrm{[F/m]}
\label{eq11.9}
\end{equation}
となることが知られている（導出は電磁気学の教科書に譲る）。
数値を入れてみよう。$a = 0.5$\,mm，$d = 5$\,mmなら
$L' = (\mu_0/\pi) \ln 11 \approx 0.96\,\mathrm{\mu H/m}$，
$C' = \pi\varepsilon_0 / \ln 11 \approx 12\,\mathrm{pF/m}$である。
寸法を変えても対数はゆっくりしか変わらないので，次の目安を覚えておくとよい。

すなわち，ふつうの太さ・間隔の配線は\textbf{長さ$1$\,mmあたり約$1$\,nH}の寄生インダクタンスと
\textbf{長さ$1$\,mあたり約$10$\,pF}の寄生キャパシタンスをもつ ― この目安は覚えておく価値がある。
なお式\ref{eq11.8}と式\ref{eq11.9}の積は$L' C' = \mu_0 \varepsilon_0 = 1/c_0^2$となって配線の形によらない。
信号が配線を光速で伝わることと対応する美しい関係である。

\subsection{寄生成分は高周波でだけ顔を出す}

$1$\,nHや$10$\,pFという値は一見どうでもよい小ささに見える。
実際，長さ$10$\,cmの配線（$L \approx 100$\,nH）のインピーダンス$|Z| = 2\pi f L$は
$50$\,Hzでは$3\times10^{-5}\,\Omega$と完全に無視できる。
ところが$30$\,MHzでは約$19\,\Omega$ ― もはや「ただの線」ではなく，れっきとした回路素子である。
スイッチングの高調波は数十MHzまで届くから，
\textbf{ノイズの周波数帯では，すべての配線がインダクタであり，向かい合うすべての導体がキャパシタである}。

\begin{注意}
寄生成分をもつのは配線だけではない。部品自身も，リード線の寄生インダクタンスと
電極間の寄生キャパシタンスをもつ。たとえばキャパシタは，寄生インダクタンスとの直列共振周波数
（\index{じこきょうしんしゅうはすう@自己共振周波数}\textbf{自己共振周波数}）より上では
逆にインダクタとして振る舞い，「コンデンサを付けたのに高周波ノイズが素通りする」ことが起こる。
\end{注意}

\begin{問}
\item 長さ$4$\,cmの配線の寄生インダクタンス（目安：$1$\,nH/mm）を見積り，
$100$\,MHzにおけるインピーダンス$|Z| = 2\pi f L$を求めよ。
\end{問}

\section{ノイズ源の等価回路モデルと伝搬経路}
\label{sec11.5}

ノイズの問題は必ず，(1)~\index{のいずげん@ノイズ源}\textbf{ノイズ源}（急峻な$dv/dt$・$di/dt$をもつスイッチング波形），
(2)~\index{でんぱんけいろ@伝搬経路}\textbf{伝搬経路}（寄生成分がつくる，回路図にない道），
(3)~\textbf{受け手}（乱される機器・回路）の3つの組合せで起こる。
対策とは，このどれかを弱めることである。本節では，源と経路を最小の等価回路で定量化する。

まず\textbf{電圧の急変}。電圧が急変するノード（たとえばチョッパのスイッチとダイオードの接続点）に
寄生キャパシタンス$C_p$がぶら下がっていると，キャパシタの基本式から
\begin{equation}
i = C_p \frac{dv}{dt}
\label{eq11.10}
\end{equation}
の電流が流れる。導体がつながっていなくても，電場の変化を介して流れるこの電流を
\index{へんいでんりゅう@変位電流}\textbf{変位電流}という。
$C_p$がどんなに小さくても，$dv/dt$が大きければ電流は大きい
（数値例は\ref{sec11.6}節の例題で示す）。

次に\textbf{電流の急変}。電流が急変する経路に寄生インダクタンス$L_p$があると，インダクタの基本式から
\begin{equation}
v = L_p \frac{di}{dt}
\label{eq11.11}
\end{equation}
の電圧が発生する。スイッチを切った瞬間に素子の両端に現れるこの跳ね上がりを
\index{さーじでんあつ@サージ電圧}\textbf{サージ電圧}という。
サージ電圧はノイズであると同時に，素子の耐圧（2章・3章）を脅かす存在でもある。

そして両者の\textbf{共振}。実際の回路では，寄生インダクタンスと寄生キャパシタンスが
直列につながったループが必ずできる。
たとえば降圧チョッパのオン期間，オフしたダイオードは
寄生キャパシタンス$C_p$をもつコンデンサとして振る舞い（2章のpn接合の空乏層容量），
配線の寄生$L_p$・$R$とともに直列RLC回路をつくる（\FIGref{fig11.5}(a)）。
ここにスイッチングの急峻な電圧が加わると，回路は固有の共振周波数
\begin{equation}
f_r = \frac{1}{2\pi \sqrt{L_p C_p}}
\label{eq11.12}
\end{equation}
で振動する。スイッチングのたびに波形に重畳するこの減衰振動を
\index{りんぎんぐ@リンギング}\textbf{リンギング}という（\FIGref{fig11.5}(b)）。
スペクトルの上では$f_r$付近を持ち上げる鋭いピークとして現れ，放射ノイズの主犯になることが多い。

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.5.eps}
\end{center}
\caption{寄生成分がつくるノイズの等価回路とリンギング。(b)は$L_p = 40$\,nH，$R = 0.1\,\Omega$，
$C_p = 0.1$\,nF，振幅$10$\,Vの場合で，ほぼ2倍まで跳ね上がり$f_r \approx 80$\,MHzで鳴り続ける}
\label{fig11.5}
\end{figure}

数値を入れてみよう。配線長約4\,cmぶんの寄生インダクタンス$L_p = 40$\,nHと
ダイオードの寄生キャパシタンス$C_p = 0.1$\,nFなら，式\ref{eq11.12}より
$f_r = 1/(2\pi\sqrt{L_p C_p}) \approx 80$\,MHz ― FMラジオ放送（$76\sim95$\,MHz）とちょうど重なる。
スイッチング電源の近くでラジオに入る雑音の正体は，
数cmの配線と1個の半導体接合がつくる，回路図にないRLC共振なのである。

\begin{問}
\item 寄生インダクタンス$100$\,nHの配線を流れる$20$\,Aの電流を$50$\,nsで遮断した。
式\ref{eq11.11}で発生するサージ電圧を見積れ。
\end{問}

\section{コモンモードノイズ}
\label{sec11.6}

伝導ノイズには，流れ方の異なる2つのモードがある。区別できることが対策の第一歩である。

\begin{定義}[ディファレンシャルモードとコモンモード]
\label{def11.4}
電源線などの往復2本の線を\textbf{互いに逆向き}に流れ，負荷・電源とつくるループを回る電流の成分を
\index{でぃふぁれんしゃるもーど@ディファレンシャルモード}\textbf{ディファレンシャルモード}
（\index{のーまるもーど@ノーマルモード}ノーマルモード），
2本を\textbf{同じ向き}に流れ，大地（グラウンド）を通って戻ってくる電流の成分を
\index{こもんもーど@コモンモード}\textbf{コモンモード}という（\FIGref{fig11.6}）。
\end{定義}

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.6.eps}
\end{center}
\caption{2つのモードの電流経路。コモンモードは対地寄生容量$C_p$と大地を通って戻り，
経路のループが大きいためアンテナとして放射しやすい}
\label{fig11.6}
\end{figure}

ディファレンシャルモードのノイズは，リプルやリンギングが通常の電流経路に乗ったものであり，
経路が回路図に描けるぶん理解しやすい。厄介なのはコモンモードである。

\subsection{発生メカニズム ― 対地寄生容量を通る変位電流}

電力変換器の中で最も激しく電位が振れるのは，スイッチ素子のドレイン（コレクタ）まわりである。
発熱するスイッチ素子は放熱板に固定されるが，放熱板は多くの場合，接地された筐体につながっている。
素子と放熱板は絶縁シートを挟んで向かい合う導体だから，そこには数十$\sim$数百pFの
\index{たいちきせいようりょう@対地寄生容量}\textbf{対地寄生容量}$C_p$ができる。
スイッチングのたびに，この$C_p$を通って変位電流（式\ref{eq11.10}）が大地へ流れ出す。
逃げた電流は接地線や電源線を通って必ず変換器へ戻ってくる ―
このとき電源の2本の線を\textbf{同じ向きに}通るから，コモンモード電流になるのである。

\begin{例題}[コモンモード電流の見積り]
\label{rei11.3}
スイッチ素子と放熱板の間の対地寄生容量を$C_p = 100$\,pFとする。
GaN素子のスイッチングにより，ドレイン電圧が$10$\,V/nsの速さで変化するとき，
大地へ流れ出すコモンモード電流を見積れ。
\begin{解答}
$dv/dt = 10\,\mathrm{V/ns} = 10^{10}\,\mathrm{V/s}$だから，式\ref{eq11.10}より
\begin{equation*}
i = C_p \frac{dv}{dt} = 100\times10^{-12} \times 10^{10} = 1\,\mathrm{A}
\end{equation*}
となる。わずか$100$\,pF ― 指先ほどの放熱板とシートの間の容量 ― を通って，
\textbf{1\,Aもの高周波電流}が回路図に存在しない経路を流れる。
主回路の電流が数Aの電源なら，ノイズ電流が主電流と同じ桁に達するということである。
\end{解答}
\end{例題}

\subsection{なぜコモンモードが最も厄介か}

\begin{箇条書}
\item \textbf{経路が回路図にない}：大地・筐体・放熱板といった
「回路図に描かれない導体」が経路になるため，設計段階で見落としやすい
\item \textbf{ループが大きく，放射しやすい}：経路は機器の外のケーブルと大地を含む
大きなループになる。導体長が波長に近づくとアンテナになる（\ref{sec11.1}節）から，
わずかなコモンモード電流でも強い放射ノイズを生む
\item \textbf{ふつうのフィルタが効かない}：ディファレンシャルモード用の対策部品は
往復電流のバランスを前提にしているため，コモンモードには専用の部品（次節）が必要になる
\end{箇条書}

\begin{問}
\item 例題\ref{rei11.3}の電源で，ゲート抵抗を大きくして$dv/dt$を$1$\,V/nsに抑えた。
コモンモード電流はいくらになるか。またこの変更はノイズを何dB減らすか。
\end{問}

\section{EMC試験と電磁ノイズ対策部品}
\label{sec11.7}

\subsection{EMCという考え方と規格}

電子機器はノイズを「出す側」であると同時に「受ける側」でもある。両方を束ねる概念がEMCである。

\begin{定義}[電磁両立性（EMC）]
\label{def11.5}
機器が，(1)~周囲に許容できない電磁妨害を与えず
（\index{えみっしょん@エミッション}\textbf{エミッション}の抑制），
(2)~周囲からの電磁妨害を受けても正しく動作する
（\index{いみゅにてぃ@イミュニティ}\textbf{イミュニティ}の確保）という2条件を同時に満たす性質を
\index{でんじりょうりつせい@電磁両立性}\textbf{電磁両立性}
（\index{いーえむしー@EMC}\textbf{EMC}：electromagnetic compatibility）という。
\end{定義}

エミッションの上限は国際規格で定められている。中心になるのは国際電気標準会議（IEC）の
特別委員会\index{しすぷる@CISPR}\textbf{CISPR}（国際無線障害特別委員会）の規格群で，
各国の規制（日本ではVCCIなど）の土台になっている。測定は大きく2つに分かれる。

\begin{箇条書}
\item \textbf{伝導エミッション試験}（おおむね$150$\,kHz$\sim$$30$\,MHz）：電源線を伝わるノイズ電圧を，
\index{ぎじでんげんかいろもう@擬似電源回路網}\textbf{擬似電源回路網}
（\index{えるあいえすえぬ@LISN}LISN）という規定インピーダンスの治具を介して測り，dB$\mathrm{\mu}$Vで規格値と比較する
\item \textbf{放射エミッション試験}（おおむね$30$\,MHz$\sim$$1$\,GHz）：
電波を反射しない\index{でんぱあんしつ@電波暗室}\textbf{電波暗室}で，規定距離のアンテナで電界強度を測る
\end{箇条書}

境目の$30$\,MHzは，このあたりから配線やケーブルがアンテナとして働き，
ノイズの主役が伝導から放射へ移ることに対応している。
規格に適合しない製品は出荷できない。設計の最終段階で試験に落ちれば基板配置からやり直しになることさえある。
だからこそ，「源と経路の物理」を設計の初期から意識することに価値がある。

\begin{COLUMN}[EMC試験は電子機器の「排ガス規制」]
自動車が排ガスの規制値を満たさなければ販売できないのと同じように，
電子機器は電磁ノイズの規制値を満たさなければ市場に出せない。
電磁ノイズは目に見えないが，無線通信や医療機器が張り巡らされた現代社会ではまぎれもない「公害」である。
2024年には国内の大手メーカーで20年以上にわたるEMC試験の不正が発覚し，大きな問題になった。
規格試験が製品開発でどれほど重い関門か ― 裏を返せば，
ノイズを定量的に設計できる技術者がどれほど求められているか ― を物語る事件である。
\end{COLUMN}

\subsection{電磁ノイズ対策部品}

市販のACアダプタを開けると，基板の入口側のかなりの面積をノイズ対策部品が占めている。
代表的な構成を\FIGref{fig11.7}に示す。どの部品がどちらのモードに効くかを対応づけて見てほしい。

\begin{figure}[h]
\begin{center}
\includegraphics[clip]{figures/fig11.7.eps}
\end{center}
\caption{EMCフィルタの代表的な構成。Xコンデンサはディファレンシャルモードを線間で短絡し，
Yコンデンサはコモンモードを大地へ逃がし，コモンモードチョークはコモンモードだけに
大きなインダクタンスとして働く}
\label{fig11.7}
\end{figure}

\begin{箇条書}
\item \index{えっくすこんでんさ@Xコンデンサ}\textbf{Xコンデンサ}：2本の電源線の\textbf{線間}に入れるキャパシタ。
ディファレンシャルモードの高周波電流の低インピーダンスな近道になり，
ノイズを電源側へ通さずその場で折り返させる
\item \index{わいこんでんさ@Yコンデンサ}\textbf{Yコンデンサ}：各電源線と\textbf{大地の間}に入れるキャパシタ。
コモンモード電流を，ケーブルの大きなループを回る前に最短経路で大地へ返す。
ただし商用周波数の\index{もれでんりゅう@漏れ電流}\textbf{漏れ電流}$i = 2\pi f C V$も
大地に流すため，感電防止の観点から容量は数nFまでに制限される
\item \index{こもんもーどちょーく@コモンモードチョーク}\textbf{コモンモードチョーク}：
1つの磁心に往復2本の線を\textbf{同じ向きに}巻いたインダクタ。
ディファレンシャルモード電流のつくる磁束は打ち消し合ってほぼゼロになる
（主回路の大電流では飽和しない）のに対し，コモンモード電流のつくる磁束は加算されるため，
コモンモードにだけ大きなインダクタンスとして働く。コモンモード対策の主役である
\item \index{ふぇらいとこあ@フェライトコア}\textbf{フェライトコア}：ケーブルにはめる筒状の磁心。
往復線をまとめて貫通させれば巻数1回のコモンモードチョークとして働き，
フェライトの磁気損失（4章）が高周波帯で抵抗として効いてノイズを熱として吸収する。
ノートパソコンの電源ケーブルの根元の「こぶ」の正体である
\end{箇条書}

対策の定石は，(1)~まず\textbf{源}を穏やかにし（ゲート抵抗やスナバによる$dv/dt$・$di/dt$の制御。
ただしスイッチング損失とのトレードオフ ― 3章），(2)~次に\textbf{経路}を断ち（レイアウトとフィルタ部品），
(3)~最後に\textbf{シールド}で閉じ込める，という順である。

\begin{問}
\item $C = 4.7$\,nFのYコンデンサを$100$\,V・$60$\,Hzの電源線と大地の間に入れた。
商用周波数の漏れ電流$i = 2\pi f C V$を求めよ
（この値が安全規格の許容値を超えないことが，Yコンデンサの容量の上限を決めている）。
\end{問}
