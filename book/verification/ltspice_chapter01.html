pn接合ダイオードのI-V特性試験回路 接続チェック OK

関連節: 1.6節 pn接合の物理 (バイアスと整流性) / ファイル: chapter01/pn_junction_iv.asc

V1をR1=1kΩ経由でD1(pn接合ダイオードモデル DMOD, Is=1e-14, N=1.0, Rs=0.1)へ直列接続し、-5V〜+1Vで.dcスイープ。ngspiceで検算した結果、逆バイアス〜低順バイアスでは電流がpAオーダーに留まり、0.6V付近から電流が急増する整流特性が確認できた(本文の「ダイオードの順方向電圧約0.7V」に対応)。

ネットリスト (asc_to_net.py 自動生成):

* netlist auto-generated by asc_to_net.py from pn_junction_iv.asc
V1 N001 0 0
R1 N001 N002 1k
D1 N002 0 DMOD
.model DMOD D(Is=1e-14 N=1.0 Rs=0.1)
.dc V1 -5 1 0.01
.end

ショットキー接合ダイオードのI-V特性試験回路 接続チェック OK

関連節: 1.7節 金属-半導体接合 (ショットキー接合とオーミック接合) / ファイル: chapter01/schottky_iv.asc

同一トポロジでD1をショットキーダイオードモデル(DSCHOTTKY, Is=1e-9, N=1.05, Rs=0.05)に置き換え。ngspice検算で順方向電圧の立ち上がりが約0.27〜0.36Vとなり、本文の「順方向電圧がpn接合より小さく(0.3〜0.4V)」と整合することを確認した。逆方向リーク電流もpn接合より大きい(nAオーダー)。

ネットリスト (asc_to_net.py 自動生成):

* netlist auto-generated by asc_to_net.py from schottky_iv.asc
V1 N001 0 0
R1 N001 N002 1k
D1 N002 0 DSCHOTTKY
.model DSCHOTTKY D(Is=1e-9 N=1.05 Rs=0.05)
.dc V1 -5 1 0.01
.end

金属-半導体オーミック接合 確認回路 接続チェック OK

関連節: 1.7節 金属-半導体接合 (オーミック接合) / ファイル: chapter01/ohmic_contact_iv.asc

接触抵抗Rc=1Ω(模式値)と半導体バルク抵抗Rb=100Ω(模式値、例題1.2の桁数そのものではなく説明用の代表値)を直列にした純抵抗回路。-1V〜+1Vの.dcスイープでngspice検算した結果、原点対称の完全な直線(I=V/101Ω)となり、整流性接触(上記2回路)と対比してオーミック接合が「どちら向きにも低抵抗」であることを示す。
