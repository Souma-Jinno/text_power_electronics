交流位相調整回路・抵抗負荷(sec10.3, ex10.5(a)(b)) 接続チェック OK

関連節: sec10.3 / ex10.5 / ファイル: chapter10/phase_control_r_load.asc

単相交流(振幅141V,50Hz)を逆並列サイリスタ(2章の2トランジスタ等価回路を2組、cathode/anodeの片側を回路グラウンド0に固定した構成)で位相制御し抵抗負荷100Ωへ与える回路。Rsrc=0.5Ω(電源線抵抗相当)。点弧角α=30/60/90/120/150degの5ケースを.param+alterparam+reset+tran(単純なalter @inst[pulse]=[...]構文はこのngspice(ngspice-41)では動作しないため不採用、alterparamでPULSE遅延の.paramを書き換えreset+tranで完全に独立な5回のトランジェント解析として実行)で1本のnetlistにまとめた。ngspice実測RMS(I(Vin)×Rload)は式10.3予測に対しα=30/60/90/120/150degそれぞれ98.77%/98.79%/98.73%/98.49%/97.60%の高い一致率(実測がわずかに低いのはRsrcでの分圧+サイリスタのオン電圧降下という物理的に妥当な系統誤差)。α=30→60degでのV_rms低下8.7Vに対しα=120→150degでは26.9Vと低下幅が3倍超に拡大し「点弧角を遅らせるほど実効値が急速に下がる」ことを定量確認。ex10.5(b)のα=90degでは実測69.608V(式10.3の理論値ちょうど70.500V=V0/2の98.73%)で「約70.5V」の主張を波形・式10.3の両方で確認。ngspice.meas RMSの結果はPythonでraw波形から直接sqrt(mean(v^2))台形積分した値と誤差0.0006%で一致し、この回路波形に対する.meas RMSの信頼性を実測で裏取りした。単一サイリスタでのAC自然転流(電圧反転で電流が自動的にゼロへ戻る挙動)を2章のモデルのまま追加改造なしで確認できたこと、および両サイリスタの端子を両方とも浮遊ノードにすると2個目の点弧時にNewton法が収束破綻する(トポロジー上、片側を回路グラウンドに固定する必要がある)ことを検証過程で発見し、tools/README.mdに追記した。

* netlist auto-generated by asc_to_net.py from phase_control_r_load.asc
Vin vin_p 0 SIN(0 141 50)
Rsrc vin_p vline 0.5
Rload vline mid 100
Q2p gbasep bbasep mid QPNP2
Q1p bbasep gbasep 0 QNPN2
Rshuntp gbasep 0 1k
Igatep 0 gbasep PULSE(0 10m {adelp} 1u 1u 1m 20m)
Q2n gbasen bbasen 0 QPNP2
Q1n bbasen gbasen mid QNPN2
Rshuntn gbasen mid 1k
Igaten mid gbasen PULSE(0 10m {adeln} 1u 1u 1m 20m)
.param adelp=1.6667m
.param adeln=11.6667m
.model QNPN2 NPN(BF=50)
.model QPNP2 PNP(BF=50)
.control
tran 10u 100m
meas tran irms_a30 RMS i(Vin) FROM=60m TO=99.9m
alterparam adelp = 3.3333m
alterparam adeln = 13.3333m
reset
tran 10u 100m
meas tran irms_a60 RMS i(Vin) FROM=60m TO=99.9m
alterparam adelp = 5.0m
alterparam adeln = 15.0m
reset
tran 10u 100m
meas tran irms_a90 RMS i(Vin) FROM=60m TO=99.9m
alterparam adelp = 6.6667m
alterparam adeln = 16.6667m
reset
tran 10u 100m
meas tran irms_a120 RMS i(Vin) FROM=60m TO=99.9m
alterparam adelp = 8.3333m
alterparam adeln = 18.3333m
reset
tran 10u 100m
meas tran irms_a150 RMS i(Vin) FROM=60m TO=99.9m
write chapter10/phase_control_r_load.raw all
.endc

交流位相調整回路・RL負荷版(sec10.3, ex10.5(c)) 接続チェック OK

関連節: sec10.3 / ex10.5(c) / ファイル: chapter10/phase_control_rl_load.asc

phase_control_r_loadのRload=100Ωに直列でLload=300mHを追加した点のみが異なる回路(α=60degで固定、R版の同αケースと比較するために作成)。負荷角phi=atan(2*pi*50*0.3/100)≈43.3deg。ngspiceのraw波形をPythonで直接読みi(Vin)のゼロクロス時刻を検出したところ、R版では電流がv(vin)の零点(定常t≈50.0003ms)とほぼ同時(遅れ検出不能なレベル)に消弧するのに対し、RL版では零点から2.3250ms=41.85deg遅れたt=52.3253msで消弧することを確認。導通区間はR版の120deg(点弧角60deg〜自然消弧180deg)に対しRL版は161.85deg(約35%拡大)で、章末問題(c)が要求する「電流が電圧の零点より遅れて消弧し、抵抗負荷のときより導通区間が広がる」という主張を定量的に確認した。実測の消弧遅れ角41.85degは負荷角phi=43.3degに近い(96.6%)が、位相制御RL回路の消弧角は超越方程式で決まり定常位相角と厳密には一致しないため、この差は物理的に妥当なものとして正直に記録し、消弧角の解析解との比較はスコープ外とした。式10.3は抵抗負荷前提のためRL負荷の実効値との数値比較は意図的に行っていない(参考値としてI_rms=0.638253Aのみ記録)。
