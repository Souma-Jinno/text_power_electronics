NMOS出力特性試験回路 接続チェック OK

関連節: sec2.5 / def2.2 / ファイル: chapter02/mosfet_output_char.asc

Vth=3V(教科書の2〜4V範囲内)のNMOSモデルで、VGS=2V(≈Vth)/3V/4V/5Vの4本の出力特性を.dcネスト掃引で取得。VGS=2V,3VはVDS全域で電流ほぼゼロ、VGS=4V,5Vは原点(0A,0V)から連続的に立ち上がり(0.7Vのようなオン電圧の下駄が無い)ことをngspiceで定量確認した。

ネットリスト (asc_to_net.py 自動生成):

* netlist auto-generated by asc_to_net.py from mosfet_output_char.asc
M1 vd vg 0 0 MYNMOS
VDS vd 0 0
VGS vg 0 2
.model MYNMOS NMOS(LEVEL=1 VTO=3 KP=2)
.control
dc VDS 0 10 0.2 VGS 2 5 1
print v(vd) v(vg) i(vds)
write chapter02/mosfet_output_char.raw
.endc
.end

IGBT等価回路（簡易・非公式マクロモデル） 接続チェック OK

関連節: sec2.6（例題2.3の定性的検証） / ファイル: chapter02/igbt_equivalent_von.asc

【注意】実IGBTデバイスの正式なSPICEマクロモデルではない。sec2.6本文の説明(縦型MOSFETの裏面n+層をp形に置き換えただけ)に基づく教育目的の簡易等価回路（MOSFETチャネル+pn接合ダイオード1個の直列)。電流掃引0.1〜60Aで、ダイオード成分(v(vmid))は0.477V→0.702Vとほぼ一定(対数的)に留まり本文の『Von型』傾向を定性的に示せたが、MOSFETチャネル分を含む合成電圧(v(vcol))は電流とともに単調に増加し続け、例題2.3の『Von≈1.25Vでほぼ一定』は再現できていない(伝導度変調を再現しない単純直列モデルの限界)。数値そのもの(1.25V,12.5W,62.5W等)の再現は意図的に行っていない。詳細はVERIFICATION_NOTES.md参照。

ネットリスト (asc_to_net.py 自動生成):

* netlist auto-generated by asc_to_net.py from igbt_equivalent_von.asc
M1 vcol N001 vmid vmid MYNMOS
D1 vmid 0 DMOD
VGG N001 0 15
Isweep 0 vcol 0.1
.model MYNMOS NMOS(LEVEL=1 VTO=3 KP=2)
.model DMOD D(Is=1e-9 N=1.0 Rs=0.001)
.control
dc Isweep 0.1 60 0.1
print v(vcol) v(vmid)
write chapter02/igbt_equivalent_von.raw
.endc
.end

サイリスタ2トランジスタ等価回路（ラッチ動作の実演） 接続チェック OK

関連節: sec2.7 / ファイル: chapter02/thyristor_two_transistor.asc

【注意】NPN+PNPのクロスカップリングによる教科書標準の2トランジスタ等価回路(非公式の簡易モデル)。Q1ベース-カソード間のRshunt=1kΩは本文にない追加要素で、理想Ebers-Mollモデルの数値的不安定性(ゲート電流ゼロでも誤ってオンに収束する問題)を避けるための工夫(実SCRのカソードショント抵抗に相当)。.tran解析でゲートパルス(PULSE、1ms〜2ms幅)前はアノード電流≈0.68mA(遮断)、パルス後はアノード電流が110.7mAへ急増し、ゲートパルスが完全に消えたt=2ms以降もt=10ms(解析終了)までオン状態(110.7mA)を維持し続けることを確認、sec2.7『ゲート電流を切ってもオンし続ける』というラッチ動作の核心を定量的に確認した。保持電流以下に絞ったときのターンオフ挙動は今回未検証(持ち越し)。

ネットリスト (asc_to_net.py 自動生成):

* netlist auto-generated by asc_to_net.py from thyristor_two_transistor.asc
VAA N001 0 12
RL N001 anode 100
Q2 gbase N002 anode QPNP2
Q1 N002 gbase 0 QNPN2
Igate 0 gbase PULSE(0 10m 1m 1u 1u 1m 100m)
Rshunt gbase 0 1k
.model QNPN2 NPN(BF=50)
.model QPNP2 PNP(BF=50)
.control
tran 10u 10m
print v(anode) v(gbase) i(vaa)
write chapter02/thyristor_two_transistor.raw
.endc
.end
