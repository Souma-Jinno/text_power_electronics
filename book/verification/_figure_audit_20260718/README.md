## 3点監査

1. **フォントが小さすぎないか**: 全回路ラベルfont-size=24px、統一済み（他章と同一規約）。OK。
2. **はみ出し・重なりがないか**:
   - **修正前**: 第2章4回路・第3章1回路で、詳細説明コメント(10行前後)が回路図の高さ全域に
     重なって配置され、部品ラベルが読めない状態だった(`switching_loss_double_pulse`の修正前版で
     顕著、`_v2_before/`参照)。
   - **修正後**: 全8回路で目視確認。タイトルコメントは回路上端より明確に上に分離。
     残存する軽微な重なり（`thyristor_two_transistor`のRshunt/Q1値ラベルが数px接触、
     `mosfet_output_char`のMYNMOSラベルがGND記号にわずかに接触）はいずれも文字の可読性を
     損なわない程度で、構造的な「文章が回路に重なって読めない」問題は解消。
3. **余白バランス（空白過多・偏り）**: 各回路とも右側に大きな空白（コメント無し領域）が残るが、
   これは`fix_viewbox.py`が最長コメント行の幅に合わせてviewBoxを広げる仕様のため
   （コメントを1行に削減した本ラウンドでviewBox幅も縮小方向へ改善）。教科書DTP側で個別に
   トリミング範囲を指定する運用は既存踏襲（電磁気本と同じ）。
