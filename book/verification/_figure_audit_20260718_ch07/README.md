# 第7章 LTspice図 3点監査（2026-07-18）

対象: `linear_reg_zener.svg` / `linear_reg_opamp.svg`（`book/figures/ltspice/chapter07/`）。
監査ツール: `tools/svg_preview.py`（変更なしで流用）。本章で新規追加した`zener`/`opamp`
シンボル（`tools/symbols/zener.asy`・`opamp.asy`）が正しくレンダリングされるかも合わせて確認した。

## 3点監査

### (i) フォントサイズ
部品ラベル・値・ディレクティブは24px統一（既存章と同一）。縮小・潰れなし。

### (ii) はみ出し・重なり
両回路とも本体（V1/V4〜R1/R2）とディレクティブブロック（`.control`〜`.endc`）の間に十分な
垂直マージンがあり重なりなし。`linear_reg_zener`は「vout」フラグラベルと「QNPN3」値ラベルが
近接するが文字同士の重なりではない（許容範囲、既存章のvfb/vout近接ラベルと同水準）。

### (iii) 余白バランス
`linear_reg_zener`は左上に主要回路がまとまり右側・下側に空白が生じるが、第1章corpus準拠の
880x680シートに収まる程度の回路規模のため既存章と同程度。`linear_reg_opamp`は7素子と本章内で
最も部品数が多く、E1(opamp)の三角シンボルが視覚的な焦点になるよう主要回路を上寄せに配置、
ディレクティブブロックとの間隔も十分。

## 新規シンボルの見え方確認
- `zener`: 通常のdiodeトライアングルに加え、カソード側バーに折れ線（"Z"風のkink）を追加し、
  プレーンなdiodeと視覚的に区別できることを確認（画像内`D1`参照）。
- `opamp`: 三角形+「+」「-」入力マーク+右端の出力ピンが明瞭にレンダリングされ、実LTspiceの
  UniversalOpamp2とは見た目が異なる（本パイプライン独自の簡易アートワーク）が、三角形という
  オペアンプの慣用記号としての可読性は確保できている（画像内`E1`参照）。
