# 第8章 LTspice図 3点監査（2026-07-18）

対象: `bridge_rectifier.svg`（`book/figures/ltspice/chapter08/`）。監査ツール: `tools/svg_preview.py`
（変更なしで流用）。

## 3点監査

### (i) フォントサイズ
部品ラベル・値・ディレクティブは24px統一（既存章と同一）。縮小・潰れなし。

### (ii) はみ出し・重なり
ブリッジ本体（D1〜D4・Rsrc・V1）とC1/R1・タイトル・ディレクティブブロックの間に十分な
マージンがあり重なりなし。1点、プレビューPNG上でD3のインスタンス名ラベルが他の3個
（D1/D2/D4は三角の直上）と見た目上わずかに違う向きで描画されているように見えたため実SVGを
`grep`で確認したところ、D3のシンボル本体自体は他と同じ`rotate(180)`（ソース.ascも4個とも
`R180`で統一、コピー誤りなし）で、ラベルテキスト側だけが`.asy`の`WINDOW`既定値由来の
`rotate(90)`という別グループで描画されているだけと判明（配線ミスではない、ラベル向きの
コスメティックな差異）。ngspice側の実測波形（正しく整流された脈流）でも接続の正当性を
独立に確認済み。

### (iii) 余白バランス
ブリッジ部が左寄り・C1/R1が右寄りで水平方向にバランスよく分散、上下の余白も既存章と同程度。

## 監査結果: NGなし（D3ラベル回転はコスメティック差異、配線に問題なし）。
