## 1. bjt_base_drive（sec2.4、例題2.2）

回路: VCC=12V - RC=0.02Ω - コレクタ、ベースへ電流源で2A固定注入（電流源はI1、R180配置で
`I1 0 nbase 2` として nbase へ電流を注入する向きに接続）、エミッタ接地。
`.model QNPN NPN(BF=50)`。`.op` を `.control`ブロック内で実行し、
`print v(ncoll) v(nbase) @q1[ic] @q1[ib] @q1[ie] @q1[vbe]` でログ出力。

**ngspice出力（実測）**:
```
v(ncoll) = 1.000000e+01
v(nbase) = 1.072011e+00
@q1[ic]  = 1.000008e+02   (100.0008 A)
@q1[ib]  = 2.000015e+00   (2.000015 A)
@q1[ie]  = -1.02001e+02
@q1[vbe] = 1.072011e+00
```

**本文との突き合わせ**: 例題2.2は「β=50のBJTでコレクタ電流100Aを流すにはiB=2Aが必要
（iC=βiB）」。実測 iC≈100.0A・iB≈2.000A・β=iC/iB≈50.0004 で **定量的に一致**。
v(ncoll)=10.0Vは VCC−iC·RC = 12−100×0.02 = 10V の手計算とも一致し、RC=0.02Ωの選定
（活性領域を保つ）が妥当だったことも確認できた。

**確認できなかった点（正直に記載）**: 例題2.2(b)の「ベース-エミッタ間電圧を0.7Vとしたときの
駆動電力P_B=1.4W」は、本文が採用した0.7Vという値そのものが手計算用の仮定であり、
今回の`.model QNPN NPN(BF=50)`（Is等はSPICEデフォルト）でのシミュレーション上のVbeは
実測1.072Vとなった（100Aという大電流をIsデフォルト値のダイオード式で受けるとVbeが
0.7Vより高く出る）。したがって「iC≈100A・iB≈2A・β≈50」という例題2.2(a)の核心は
シミュレーションで再現できたが、(b)のP_B=1.4Wという数値そのものはVbeモデル依存であり、
このシミュレーションでは再現・検証していない（本文の0.7V仮定は独立した簡略化として扱う）。

## 2. mosfet_output_char（sec2.5、fig2.5の概念・def2.2 Vth=2〜4V）

回路: `.model MYNMOS NMOS(LEVEL=1 VTO=3 KP=2)`（Vth=3V、教科書の「パワーMOSFETで2〜4V程度」の
範囲内）。ドレインにVDS、ゲートにVGSをそれぞれ電圧源で与え、ソース接地（ボディもソースに
同電位で接続）。`.dc VDS 0 10 0.2 VGS 2 5 1` のネスト掃引でVGS=2/3/4/5Vの4本の出力特性を取得。

**ngspice出力（実測、抜粋）**:
- VGS=2V（=Vth）: 全VDS域で `i(vds)` はおよそ -1e-14〜-1e-11A（実質ゼロ、ソフトウェア誤差程度）
- VGS=3V: 同様にほぼゼロ（Vth未満なのでオフ）
- VGS=4V: VDS=0で0A、VDS=0.2Vで0.36A、VDS=1.0V以降は1.00Aで飽和（原点から滑らかに立ち上がる）
- VGS=5V: 同様の形状でVDS=1.0V以降は4.00Aで飽和

**本文との突き合わせ**: 本文sec2.5「ダイオードやBJTと違い、特性が原点からまっすぐ立ち上がる」
「0.7Vのようなオン電圧の下駄がない」という主張について、VGS=4V/5Vの特性がVDS=0の原点
（電流0A）から連続的に（0.7Vのような不連続なオフセットなしに）立ち上がることを**定量的に確認**
できた。またdef2.2「しきい値電圧Vth（パワーMOSFETで2〜4V程度）」についても、VGS=Vth=2Vちょうど
では電流がほぼゼロ（オン・オフの境界）であることを確認し、設定したVTO=3Vが本文の2〜4V範囲内に
収まっていることも満たしている。

## 3. igbt_equivalent_von（sec2.6、例題2.3の定性的検証）

**重要な前提（誤解防止のため必ず明記）**: この回路は実IGBTデバイスの正式なSPICEマクロモデルでは
ない。sec2.6本文の説明「縦型MOSFETの裏面のn+層をp形に置き換えただけ」に基づく**教育目的の簡易
等価回路（非公式）**であり、MOSFETチャネル1個（M1, `.model MYNMOS NMOS(LEVEL=1 VTO=3 KP=2)`、
ゲートはVGG=15VでVth=3Vより十分高く固定しオン状態を維持）と、追加のpn接合1個分の電圧降下
（D1, `.model DMOD D(Is=1e-9 N=1.0 Rs=0.001)`、Vf≈0.48〜0.70V程度になるよう調整）を**直列**に
配置しただけのものである。`.dc Isweep 0.1 60 0.1` で電流を0.1A〜60Aまで掃引し、
`v(vcol)`（M1ドレイン=等価コレクタ、Isweepから見た全体のVon）と
`v(vmid)`（M1ソース/D1アノード、MOSFETチャネルとダイオードの中間点＝ダイオード単体の電圧）を観測。

**ngspice出力（実測、抜粋）**:
```
I=0.1A:  v(vcol)=0.481V  v(vmid)=0.477V  (チャネル分=vcol-vmid≈0.004V)
I=1.0A:  v(vcol)=0.581V  v(vmid)=0.537V  (チャネル分≈0.044V)
I=10A :  v(vcol)=1.053V  v(vmid)=0.606V  (チャネル分≈0.447V)
I=25A :  v(vcol)=1.804V  v(vmid)=0.644V  (チャネル分≈1.160V)
I=50A :  v(vcol)=3.169V  v(vmid)=0.687V  (チャネル分≈2.482V)
I=59.9A: v(vcol)=3.769V  v(vmid)=0.702V  (チャネル分≈3.067V)
```

**本文との突き合わせ（定性的、正直な範囲で）**:
- ダイオード成分`v(vmid)`は 0.1A→59.9A（3桁弱の電流変化）に対し 0.477V→0.702V と
  約0.22Vしか変化しない（対数的にしか増えないダイオードの性質どおり）。これは低電流域でも
  すでに約0.48Vのオフセットが乗っており、回路2（MOSFET単体）が低電流でほぼ0Vに漸近するのとは
  対照的な「Von型（電流によらないオフセットを持つ）」性質を確認できた。
- 一方、直列に入れたMOSFETチャネル分（vcol−vmid）は電流とともに単調に増加し続け
  （0.004V→3.07V）、この簡易等価回路の合成電圧v(vcol)全体も電流とともに増加を続ける。
  **これは教科書の例題2.3が示す「IGBTのオン電圧はV_on=1.25V程度でほぼ一定（電流によらない）」
  という主張を、この簡易モデルは再現できていない**。理由は正直に記す:
  実IGBTでは伝導度変調（def2.3）によりドリフト層の抵抗そのものが電流とともに下がるため
  高電流でもRon型の増加が起こりにくいが、本回路はSPICEの通常のMOSFETモデル（伝導度変調を
  再現しない固定Ronのチャネル）をそのまま直列に使っているため、この効果を再現できない。
- したがって本回路で確認できたのは「ダイオード成分が電流によらずほぼ一定」という定性的傾向のみで
  あり、例題2.3の数値（1.25V、10A→12.5W、50A→62.5W）そのものの再現は**意図的に行っていない**
  （課題文の指示どおり）。全体の合成Von（v(vcol)）が実IGBTのようにほぼ一定になる、という主張は
  **この簡易モデルでは確認できなかった**点を明記する。

## 4. thyristor_two_transistor（sec2.7、ラッチ動作）

回路: Q2=PNP（アノード側）、Q1=NPN（カソード側）の2トランジスタ等価回路。
Q2エミッタ=アノード、Q2コレクタ→Q1ベース、Q1コレクタ→Q2ベース（クロスカップリング）、
Q1エミッタ=カソード(0V)。主回路VAA=12V - RL=100Ω - アノード。
ゲートはQ1ベースへ `PULSE(0 10m 1m 1u 1u 1m 100m)`（1ms後に1ms幅で10mA注入、以降105msまでは
不使用のため今回の10ms解析内では1回のみ）を注入。`.tran 10u 10m` で解析。

**モデル化上の工夫（本文にはない追加要素、正直に記載）**: Q1ベース-カソード間に
Rshunt=1kΩを追加した。理想Ebers-MollモデルのままだとRshuntなしでは正帰還ループが
数値的に不安定で、ゲート電流ゼロの初期時点でも演算誤差からオン状態に収束してしまい
「オフ状態からゲートパルスでオンする」という挙動を再現できなかった。Rshuntは実際の
SCRが持つ「カソードショント抵抗」（微小リーク電流を迂回させ誤点弧を防ぐ設計）に相当する
役割で、教科書本文には明記されていないモデル化上の追加だが、回路の質的な主張
（ゲートでオン・オフを切ってもオンし続ける）自体は変えていない。

**ngspice出力（実測）**:
```
t=0ms（ゲートパルス前）:      v(anode)=11.932V   i(vaa)=-6.759e-4 A（≈0.68mA、遮断状態）
t=1.000067ms（パルス立上り直後）: v(anode)=0.934V   i(vaa)=-1.10656e-1 A（≈110.7mA、オン）
t=1.09〜2.00ms（パルス継続中）:  v(anode)=0.935V   i(vaa)=-1.10650e-1 A（オン持続）
t=2.001ms（パルス消滅直後、fall 1us経過後）: v(anode)=0.935V  i(vaa)=-1.10650e-1 A（オンのまま！）
t=4.1〜9.3ms（パルス消滅から2〜7ms後）:      v(anode)=0.934V  i(vaa)=-1.10657e-1 A（オン持続）
t=10ms（解析終了時点）:        v(anode)=0.934V   i(vaa)=-1.10657e-1 A（オン持続）
```

**本文との突き合わせ**: sec2.7「いったんオンしたサイリスタは、ゲート電流を切ってもオンし続ける」
というラッチ動作の核心的主張を、**定量的に確認**できた。ゲート電流は t=1ms〜約2ms（PULSE幅1ms、
fall 1us）しか流れていないが、アノード電流はt=1.00007ms付近で0.68mA→110.7mAへ急峻に立ち上がり、
ゲートパルスが完全に消滅したt=2ms以降もt=10ms（パルス消滅から8ms後）までアノード電流が
110.7mAでほぼ一定のまま流れ続けることを確認した。遮断状態（t=0、0.68mA）とオン状態
（110.7mA）の電流比は約163倍であり、明確な状態遷移として区別できる。

**確認できなかった点（正直に記載）**: 本文が説明する「保持電流以下まで主回路電流を減らせば
オフに戻る」というターンオフ機構は、今回の10ms解析では意図的に主回路電流を絞る操作を
行っていないため検証していない（持ち越し。必要であれば別途VAAまたはRLを操作してターンオフ
挙動を追加検証できる）。またQNPN2/QPNP2ともBF=50のみを指定した理想モデルであり、実サイリスタの
保持電流やラッチ電流の実測値との数値的対応は取っていない（定性的なラッチ動作の確認が目的）。

## 全体所感・妥協点まとめ

- 回路1・2は本文の数値（β=50・iB=2A・iC=100A、Vth=2〜4V範囲）と**定量的に一致**を確認できた。
- 回路3・4は教科書に明記のないモデル化上の工夫（IGBT等価回路の簡易性、サイリスタのRshunt）を
  加えており、それぞれ検証メモ中に「確認できた点」と「確認できなかった点／再現できなかった点」を
  正直に分けて記載した。
- 回路3は特に、「合成Von(v(vcol))が電流によらずほぼ一定」という教科書の主張（例題2.3の骨子）を
  この簡易モデルでは**再現できなかった**ことを隠さず明記している。ダイオード単体の電圧
  （v(vmid)）は本文の主張と整合する定性的傾向（電流によらずほぼ一定）を示したが、
  MOSFETチャネル分を直列に含む全体としての合成Vonまでは再現できていない。
